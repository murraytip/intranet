# Manual *in situ* Detection

We can help with detection methods including, but not limited, to:

- Immunohistochemistry
- Immunofluorescence
- Enzyme Histochemistry
- TUNEL
- Phalloidin staining
- Whole mount embryo staining
- IF staining of cell organelles
